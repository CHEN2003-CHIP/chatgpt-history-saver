# Contributing

Thanks for your interest in improving ChatGPT History Saver.

## Ways to contribute

- Report bugs with clear reproduction steps
- Suggest export improvements or UI polish
- Improve compatibility with changing ChatGPT page structures
- Help refine documentation and release materials

## Development flow

1. Fork the repository
2. Create a feature branch
3. Make focused changes
4. Reload the unpacked extension in Edge
5. Test `Export HTML` and `Export PDF` on real ChatGPT conversations
6. Open a pull request with a clear summary

## Guidelines

- Keep changes small and reviewable
- Prefer clear behavior over clever abstractions
- Preserve user privacy and avoid introducing any network transmission of conversation data
- Update docs when behavior changes

## Pull request checklist

- The extension still loads in Edge
- HTML export was tested on a real conversation
- PDF export was tested on a real conversation
- Any user-facing change is reflected in the README or docs

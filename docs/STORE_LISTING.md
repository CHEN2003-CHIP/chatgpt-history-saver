# Store Listing Draft / 商店资料草案

## Product name / 产品名称
ChatGPT History Saver

## Short description (EN)
Export ChatGPT conversations to high-fidelity HTML snapshots and print-friendly PDFs.

## 简短描述（中文）
将 ChatGPT 对话导出为高保真 HTML 静态网页和适合打印保存的 PDF。

## Full description (EN)
ChatGPT History Saver helps you save ChatGPT conversations in formats that are easy to read, archive, and share.

Use the extension popup on a ChatGPT conversation page to:
- Export a print-friendly PDF preview
- Export a high-fidelity HTML snapshot for offline reading and archiving

The extension works locally in the browser and only runs when you explicitly export a conversation.

## 完整描述（中文）
ChatGPT History Saver 可以帮助你把 ChatGPT 网页对话保存为更易阅读、归档和分享的格式。

你可以在 ChatGPT 对话页面点击扩展弹窗，然后：
- 导出适合打印保存的 PDF
- 导出适合离线阅读和归档的高保真 HTML 静态网页

插件主要在浏览器本地运行，只有在你主动执行导出时才会处理当前对话内容。

## Key features / 核心功能
- One-click export from the Edge toolbar
- 从 Edge 工具栏一键导出
- High-fidelity HTML snapshots for offline reading
- 高保真 HTML 静态网页导出，适合离线阅读
- Print-friendly PDF export flow
- 适合打印保存的 PDF 导出流程
- Best-effort image embedding
- 尽量内嵌图片资源，提升导出完整度
- Works on `chatgpt.com` and `chat.openai.com`
- 支持 `chatgpt.com` 和 `chat.openai.com`

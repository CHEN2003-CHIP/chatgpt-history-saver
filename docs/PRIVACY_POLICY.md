# Privacy Policy / 隐私政策

Last updated / 最后更新: 2026-03-28

ChatGPT History Saver processes data only to export the current ChatGPT conversation that the user explicitly chooses to save.
ChatGPT History Saver 仅在用户主动触发导出时处理当前 ChatGPT 对话内容。

## What the extension accesses / 插件会访问什么

- The current active tab when the user is on `chatgpt.com` or `chat.openai.com`
- 用户位于 `chatgpt.com` 或 `chat.openai.com` 时的当前活动标签页
- The visible chat content needed to build PDF or HTML exports
- 生成 PDF 或 HTML 导出所需的可见聊天内容
- Temporary in-memory/session storage used to pass export data between extension pages
- 用于在扩展页面之间传递导出数据的临时内存或会话存储

## What the extension does not do / 插件不会做什么

- It does not transmit conversation content to any external server owned by the developer.
- 不会将聊天内容发送到开发者拥有的外部服务器。
- It does not sell, share, or monetize exported conversation data.
- 不会出售、共享或商业化用户导出的聊天数据。
- It does not create a remote account system.
- 不会创建远程账户系统。
- It does not run background analytics or advertising scripts.
- 不会运行后台分析或广告脚本。

## Data storage / 数据存储

- Conversation data is processed locally in the browser.
- 聊天数据仅在浏览器本地处理。
- Temporary export payloads are stored only long enough to open the export/preview page.
- 临时导出数据只会保留到导出或预览页面成功打开为止。
- Exported files are saved only when the user explicitly chooses to export.
- 只有在用户明确执行导出时才会保存文件。

## Permissions rationale / 权限说明

- `activeTab`: identify and work with the currently open ChatGPT tab
- `activeTab`：识别并处理当前打开的 ChatGPT 标签页
- `scripting`: inject the content collector when needed
- `scripting`：在需要时注入内容采集脚本
- `storage`: pass temporary export payloads between extension contexts
- `storage`：在扩展上下文之间传递临时导出数据
- `downloads`: allow the user to save exported HTML files
- `downloads`：允许用户保存导出的 HTML 文件
